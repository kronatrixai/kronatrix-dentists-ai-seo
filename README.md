## Deployment steps
1. Create GitHub repo: kronatrix-dentists-ai-seo.
2. Upload all files to the root of the repo.
3. Enable GitHub Pages from the main branch and root folder.
4. Add DNS CNAME: Host dentists; points to your GitHub Pages domain.
5. Set custom domain in GitHub Pages: dentists.kronatrix.co.uk.
6. Confirm the CNAME file exists and contains only dentists.kronatrix.co.uk.
7. Test homepage, /robots.txt, /sitemap.xml and /llms.txt.
8. Submit sitemap in Google Search Console and request indexing.
9. Add the site to the KRONATRIX authority network.
